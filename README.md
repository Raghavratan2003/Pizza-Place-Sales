# 🍕 Pizza Place Sales Analysis

## Business Problem
A pizza restaurant wants to understand its 2015 sales performance 
and get actionable recommendations to grow revenue.

## Dataset
Maven Analytics — Pizza Place Sales
4 tables: orders, order_details, pizzas, pizza_types

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Key Findings
- Total Revenue: $817,860
- Busiest Day: Friday
- Peak Hours: 12 PM
- Best Seller: Classic Deluxe Pizza
- Worst Seller: Brie Carre Pizza
- Highest Revenue Category: Classic, Large size

## Recommendations
1. Promote Classic Deluxe Pizza with combos/discounts
2. Discontinue Brie Carre Pizza — only ~500 orders in full year
3. Increase staff on Fridays at 12 PM — peak demand window
4. Focus marketing on Large Classic pizzas — highest revenue per order
